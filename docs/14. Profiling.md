1. Install Graphviz

- macOS: brew install graphviz
- Linux: sudo apt-get install graphviz

2. Start the server

```
cd cmd
go run main.go
```

3. Start profiling

```
go tool pprof http://localhost:6060/debug/pprof/profile\?seconds=20
```

4. Inside pprof CLI:

- type `web` to see the visualization
